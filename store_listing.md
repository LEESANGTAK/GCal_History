# 구글 캘린더 제목 자동완성 (G-Cal History) 상세 설명

## 📌 앱 개요
구글 캘린더에서 새 일정을 추가할 때, 이전에 작성했던 유사한 일정 제목과 장소, 내용을 자동으로 추천해주는 스마트한 확장 프로그램입니다.

## ✨ 주요 기능
1. **내 캘린더 기반 자동완성:** 사용자의 구글 캘린더에 연동된 실제 과거 일정(최대 1년치)을 분석하여, 제목 입력 시 드롭다운 형태로 유사한 일정들을 추천합니다.
2. **다중 캘린더 지원:** '기본 캘린더'뿐만 아니라 '기념일', '업무용' 등 사용자가 포함된 모든 종류의 캘린더에서 내역을 가져옵니다.
3. **스마트 자동 동기화:** 추천된 이전 일정을 클릭하면,
   - 이전 일정의 "제목"과 "위치/설명"이 자동 기입됩니다.
   - 해당 일정이 속했던 캘린더 종류(예: Private, 기념일)를 찾아 **캘린더 선택기(Dropdown)까지 자동으로 변경**해줍니다.
4. **완벽한 프라이버시 보호:** 모든 과정은 사용자의 브라우저 내에서만 이루어지며, 외부 서버로 일정을 전송하거나 별도로 수집하지 않습니다.

## 🚀 사용 방밥
1. 구글 캘린더에 접속합니다.
2. 빈 칸을 클릭하여 일정 생성 팝업을 띄웁니다.
3. 최초 1회, 확장 프로그램에서 본인의 구글 계정 [캘린더 읽기 권한] 승인 창을 띄웁니다. 권한을 승인해 주세요.
4. 일정 제목에 예전에 참석했던 미팅이나 기록을 한 두 글자 적어보세요.
5. 아래에 드롭다운으로 추천되는 일정을 누르면 시간 절약 끝!

## 🔐 권한에 대한 안내
본 확장 프로그램은 원활한 기능 제공을 위해 구글 캘린더 API(`calendar.readonly` 스코프)를 사용합니다. 데이터는 오직 일정 추천과 기입 목적으로만 브라우저 내에서 캐싱되며 외부 인터넷으로 유출되지 않습니다. 자세한 내용은 개인정보처리방침을 확인해주세요.
